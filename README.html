birddog

The goal of birddog is sniffing out emergence and trajectories in scientific and patent literature.

Installation

You can install the development version of birddog from GitHub with:

# install.packages("devtools")
library(devtools)

# install and load getLattes
devtools::install_github("roneyfraga/birddog")
library(birddog)
